# Particle Studio

Generative particle art on HTML Canvas — **pure vanilla JavaScript**, no framework.

## Features

- Linked particle system with mouse attract / repel modes
- Live sliders for count, speed, size, links, and trail fade
- Presets: nebula, fireflies, ocean, glitch
- PNG screenshot export

## Run

```bash
npm install
npm run dev
```

## Stack

Vanilla JS · Canvas 2D · requestAnimationFrame · Tailwind CSS v4 · Vite
